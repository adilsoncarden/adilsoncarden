## 🧭 About Me

I'm a **Full Stack Developer in continuous learning**, driven by curiosity and the desire to understand how technology can organize, simplify and improve the way we solve problems.

I don't come from years of industry experience yet, but I do come from a clear conviction:  
**improve every day**, write better code than yesterday and build systems that make sense in the long run.

I believe in a more **systematic, efficient and understandable** world, where software doesn't complicate things, but rather orders them.  
I learn by developing, making mistakes, iterating and starting again.

My current focus is on:

- understanding the fundamentals well
- writing clear code before "clever" code
- connecting frontend and backend with judgment
- growing step by step, with solid foundations

> 💡 "Consistency transforms small daily efforts into great results."
